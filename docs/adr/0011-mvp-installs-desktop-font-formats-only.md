# MVP installs desktop font formats only

Fontbrew's MVP installs and activates desktop font formats such as `.ttf`, `.otf`, `.ttc`, and `.otc`. It ignores web-only formats such as `.woff` and `.woff2` for activation, even when provider APIs expose them, because Fontbrew manages the user's system font environment rather than webfont dependencies. When multiple desktop formats are available, Fontbrew defaults to OTF preference, allows users to configure the global format preference, and allows per-install overrides such as choosing OTF or TTF explicitly. Fontbrew applies that preference even when the available desktop formats do not expose identical family/style/weight coverage.
