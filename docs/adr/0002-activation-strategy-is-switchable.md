# Activation strategy is switchable

Fontbrew activates installed packages with a strategy-based activation layer. The default strategy is `copy`, which writes tracked real font files into the Fontbrew-owned activation directory at `~/Library/Fonts/Fontbrew`. `symlink` remains supported for older manifests and explicit user configuration, but it is not the default because symlinked fonts under `~/Library/Fonts/Fontbrew` were not reliably visible to macOS font discovery. Fontbrew manages only activation artifacts recorded in its manifest and does not adopt, overwrite, or remove non-managed fonts outside that tracked artifact boundary.
